# ruby-cli-sieve-functional-prog

## Description
Given a number find all primes. Follows functional programming practices.

## Tech stack
- ruby 2.4

## Docker stack
- alpine:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
